# ImpotApp

Application Android permettant de calculer le montant des impôts locaux en fonction de plusieurs critères.

---

## Objectif

Cette application permet de calculer automatiquement :

* La surface de la maison (en m²)
* Le nombre de pièces
* La présence ou non d’une piscine

---

## Fonctionnement

L’application calcule :

* Impôt de base = surface × 2
* Supplément = (nombre de pièces × 50) + (100 si piscine)
* Total = impôt de base + supplément

---

## Interface

L’utilisateur peut :

* saisir la surface
* entrer le nombre de pièces
* cocher la case piscine
* cliquer sur le bouton "Calculer"

Le résultat s’affiche directement à l’écran.

---

## Technologies utilisées

* Java
* Android Studio
* XML (interface utilisateur)

---

## Lancer le projet

1. Cloner le projet :

```bash
git clone https://github.com/EllaVanessa/ImpotApp.git
```

2. Ouvrir avec Android Studio
3. Lancer l’émulateur
4. Cliquer sur Run
